Linguagem de Programçao: Java

Observações:

Para testar no URI Online devi-se retirar o nome do pacote que está localizado na 1º linha do código.
A plataforma não aceita o uso de package.
No uso da linguagem java o nome da class necessarimente precisa ser nomeada como <Main>
