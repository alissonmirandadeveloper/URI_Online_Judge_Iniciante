Resoluções categoria Iniciante URI Online Judge
Autor: Alisson Miranda
Linguagem de Programçao: Java

Observações:

Para testar no URI Online deve-se retirar o nome do pacote que está localizado na 1º linha do código.
A plataforma não aceita o uso de package.
No uso da linguagem java o nome da class necessarimente precisa ser nomeada como <Main>

Programming Language: Java

Comments:

To test at URI Online, you must remove the name of the package that is located on the 1st line of the code.
A platform does not accept the use of a package.
In the use of java language the class name needs to be named <Main>
